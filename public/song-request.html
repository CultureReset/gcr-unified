How the Payment Parser Works

Your request is saved and a unique reference code (like REQ-AB12CD) is generated.
You send the amount via Venmo or Cash App with the code pasted into the note/memo field.
The artist forwards the payment confirmation email to CyberCheck — the parser reads it automatically.
The parser matches your code to your request and flips it to Verified in the live queue above.
